## Register a new user




> Example request:

```bash
curl -X POST \
    "http://quiz.siit.ro/api/register" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json" \
    -d '{"email":"totam","name":"quis","password":"adipisci","password_confirm":"perspiciatis","device_name":"accusamus"}'

```

```javascript
const url = new URL(
    "http://quiz.siit.ro/api/register"
);

let headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "totam",
    "name": "quis",
    "password": "adipisci",

    "device_name": "accusamus"
}

    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
```

## Displays the users list.

<small class="badge badge-darkred">requires authentication</small>



> Example request:

```bash
curl -X GET \
    -G "http://quiz.siit.ro/api/users" \
    -H "Authorization: Bearer {YOUR_AUTH_KEY}" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json"
```

```javascript
const url = new URL(
    "http://quiz.siit.ro/api/users"
);

let headers = {
    "Authorization": "Bearer {YOUR_AUTH_KEY}",
    "Content-Type": "application/json",
    "Accept": "application/json",
};


fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());
```


> Example response (401):

```json
{
    "message": "Unauthenticated."
}
```
